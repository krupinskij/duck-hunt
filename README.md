# Duck Hunt

Duck Hunt game written in Angular

## Website

https://krupinskij.github.io/duck-hunt/

## Installation

Download or clone repository

Run

```bash
# installing dependances
$ npm install
```

## Running the app

```bash
# development
$ npm start
```
